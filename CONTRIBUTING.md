# Contributing

Thank you for your interest in contributing to this project. All contributions that improve accuracy, usability, or documentation are welcome.

## Ways to Contribute

- **Bug reports**: If you find incorrect calculations, broken links, or display issues, open an issue with steps to reproduce.
- **Data updates**: If fee rates, platform costs, or benchmark data have changed, open a PR with your sources.
- **Documentation**: Improve explanations, add examples, or fix typos.
- **Translations**: If you can translate the tool or documentation accurately, open an issue to discuss scope.

## Getting Started

This project requires no build step. The tool is a single self-contained HTML file.

1. Fork this repository
2. Make your changes to `index.html` or the relevant documentation file
3. Test in a browser before submitting
4. Open a pull request with a clear description of what changed and why

## Code Style

- Keep all logic in the single `index.html` file (no external JS dependencies)
- Match the existing CSS variable and naming conventions
- No em dashes or en dashes in visible text (use "to" for ranges, commas for pauses)
- No console.log statements

## Data Sources

When updating cost or rate data, please include a source in your PR description. Acceptable sources include:

- Official platform pricing pages (Shopify, AWS, Stripe, etc.)
- Industry analyst reports (Gartner, IDC, Forrester)
- Government or standards body publications
- Peer-reviewed research

## Reporting Issues

Open a GitHub Issue with:
- What you expected to see
- What you actually saw
- Browser and operating system
- Steps to reproduce
