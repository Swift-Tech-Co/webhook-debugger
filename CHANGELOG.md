## [1.0.0] - 2026-05-26

### Added
- Initial release of the self-contained interactive tool
- Full calculation logic matching the live tool at swifttechco.com
- Swift Tech Co. brand design system (Playfair Display, DM Sans, gold/dark palette)
- Mobile-responsive layout with CSS Grid
- Related resources section linking to comparisons, guides, and service pages
- About section with service overview and contact call to action
- MIT license

### Technical Notes
- Zero external JavaScript dependencies
- No build step required
- Single `index.html` deployable directly to GitHub Pages
