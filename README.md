Developer tool for inspecting, replaying, and debugging webhook payloads. Captures incoming requests with full headers and body, displays them in a readable format, and supports replaying to a target URL.

Useful during integration work with Stripe, GitHub, Shopify, Twilio, or any webhook-based API.

**Languages:** Node.js / TypeScript, Go
